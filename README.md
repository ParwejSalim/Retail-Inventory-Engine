# Retail-Inventory-Engine

## Description
A Java-based inventory management system designed to optimize product lookups and stock management using Object-Oriented Programming principles.

## Key Features
- Efficient product searching using HashMap for $O(1)$ average time complexity.
- CRUD operations for inventory management.
- Built using core Java to ensure high performance and scalability.

## Technologies Used
- Java
- OOP Concepts
- HashMap Data Structure

## How to Run
1. Ensure you have Java installed.
2. Compile the files: `javac SupermarketSystem.java`
3. Run the program: `java SupermarketSystem`
